## Run on Windows & Linux from source
**Required:**
- NodeJS v14.15.3
- Python 3
- Git

**On linux install `binutils` if you want to build the source code:**
```
$ sudo apt-get install -y binutils
```

**To run it development mode, do:**
```
$ npm i
$ npm run dev
```
